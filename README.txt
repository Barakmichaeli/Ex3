
*כתובת - http://localhost:8080

שאלות:
1.  המטרה של האתר הינה לשמור רשימה של מאכלים שכל בן אדם יכול להביא לאירוע.
תפעול האתר פשוט - כל מי שרוצה להביא מאכל צריך להרשם דרך עמוד ה Register  ולהכנס לעמוד הראשי
דרך הLogin.
לאחר מכן, יכול כל אחד לבחור מאכל מתוך רשימה מוגדרת ולהכניס את הID שלו (כפי שנדרש בתרגיל).
כמו כן ניתן לראות את הרשימה עד כה,  למחוק מאכל בעזרת ID או לראות מה ID מסוים מביא לאירוע.

2. עיקר הקושי מבחינתנו בעבודה היה ראשית להבין איך להעביר קבצי JSON בbody של הבקשה ואופן הכתיבה/קריאה שלהם לפי פורמט מתאים
לרשימות.
ובנוסף להתרגל למעברים בין העבודה על הUI וקובץ הtest.js שלו לבין הכתיבה של השרת בקובץ app.js והתאמה
של בקשה ותגובה לפי על מקרה.
